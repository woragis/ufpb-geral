# Fase 9 — Painel admin

## Objetivo

Criar/editar exercícios e seeds curadas sem editar código TypeScript.

## Escopo MVP

- Rota `/admin` protegida por `ADMIN_TOKEN` (env).
- Listar tópicos e seeds em destaque.
- Adicionar seed curada ao catálogo.
- Preview ao vivo.

## Fora de escopo v1

- Editor visual de solvers
- Multi-usuário / RBAC

## Critério de aceite

- Admin adiciona seed curada → aparece no catálogo da disciplina.
